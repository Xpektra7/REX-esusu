# ADR 002: Invite-Only Circles (No Community/Public Discovery)

**Date:** 2026-07-17

## Context

CircleFunds offers "community thrifts" — discoverable public circles new users can join. Our app currently only supports invite-code-based joining.

## Decision

- Circles remain **invite-only**. No public discoverability or browse feature
- This matches how real Ajo/Esusu works offline — membership is by invitation from a trusted person
- New users get their first circle via a friend's invite code

## Consequences

- Higher trust per circle (every member was vouched for)
- No viral/new-user acquisition from public listings
- Growth depends entirely on referral chains
- Less customer-support load from stranger disputes
